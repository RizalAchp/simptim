# SIMPTIM
simple timer on terminal using rust


## HOW TO RUN

- install rust toolchains 
```console
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

- clone this repo
```console
git clone --depth=1 https://github.com/RizalAchp/simptim.git
```

- build 
```console
cargo build --release
```

- run 
```console
cargo run --release
```
